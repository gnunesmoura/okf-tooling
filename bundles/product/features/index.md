# Product Features

Durable user-visible capabilities of the `tooling` library and CLI.

## Product role

Feature documents describe what the product offers and what users should be
able to observe. They are not implementation plans and do not own the
execution status of a current change.

## OKF Features

- [Feature - Summarized OKF Navigation](summarized-okf-navigation.md) - Guide for the first summarized tree reading journey.
- [Feature - OKF Concept List](okf-concept-list.md) - Plan for the concept inventory command and its filters.
- [Feature - OKF Show](okf-show.md) - Plan for the canonical single-concept read path.
- [Feature - OKF Validation](okf-validation.md) - Plan for non-blocking OKF bundle conformance reporting.
- [Feature - OKF Health](okf-health.md) - Plan for compact OKF bundle status signals after validation.
- [Feature - OKF Links](okf-links.md) - Plan for outbound link discovery and classification.
- [Feature - OKF Backlinks](okf-backlinks.md) - Plan for the concept inbound reference command.
